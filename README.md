### 制作背景
前作、[ヌーン](#ヌーン)ではPCで多くの人に遊んでもらえたが、高評価をしてくれるPCを持っていない身近な人に遊んでもらうのは難しかった。
そのためターゲットにライト層も含め、前作の操作が難しかったという点を考慮してPCでのマウス操作とスマホでのタッチ操作にに変更した。

また、制作チームはゲーム制作経験者だけでなく、アプリ開発経験者、競技プログラミング経験者などを***多様なメンバー***引き入れた。


### 担当分野
企画、プログラム、グラフィック等

### 使用技術
Unity, Plastic SCM
